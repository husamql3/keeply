# Feature Specification

**Version 1.0 · February 2026**

> ★ = Premium feature (requires paid subscription)

---

## 1. Authentication

| Feature | Description | Tier |
|---|---|---|
| Google OAuth | One-click sign-in via Google account | Free |
| GitHub OAuth | One-click sign-in via GitHub account | Free |
| Magic Link | Passwordless login via email link | Free |

---

## 2. Bookmark Addition

| Feature | Description | Tier |
|---|---|---|
| Browser Extension / Bookmarklet | One-click save from any webpage | Free |
| Manual Entry Form | URL, title, description fields for manual creation | Free |
| Duplicate Detection | Warns user when a URL already exists in their library | Free |

---

## 3. Bookmark Properties

| Property | Description | Tier |
|---|---|---|
| Title | Editable title for the bookmark | Free |
| URL | The canonical saved URL | Free |
| Description / Notes | Free-form notes field | Free |
| Date Added | Auto-stamped on creation | Free |
| Source Platform Indicator | Tags bookmark with originating platform | Free |
| Custom Tags | User-defined tags for flexible categorization | Free |
| Pin | Pin bookmark to top of its collection | Free |

---

## 4. Collections System

| Feature | Description | Tier |
|---|---|---|
| Create Collection | Name and description required on creation | Free |
| Nested Collections | Up to 3 levels of sub-collections | Free |
| Collection Cover Images | Custom or auto-generated covers from bookmarks | Free |
| Collection Stats | Total bookmarks, last updated, collaborator count | Free |

---

## 5. Organization Features

| Feature | Description | Tier |
|---|---|---|
| Grid View | Card-based layout with thumbnails | Free |
| List View | Compact, text-focused layout | Free |
| Kanban View | Column-based drag-and-drop board | Free |
| Drag-and-Drop | Reorder bookmarks within a collection | Free |
| Move Between Collections | Relocate bookmarks across collections | Free |

---

## 6. Tag Features

| Feature | Description | Tier |
|---|---|---|
| Multi-tag Support | Add multiple tags per bookmark | Free |
| Tag Management | Rename, merge, and delete tags globally | Free |
| Color-coded Tags ★ | Assign colors to tags for visual organization | **Premium** |

---

## 7. Pinning & Prioritization

| Feature | Description | Tier |
|---|---|---|
| Pin Bookmarks | Pin bookmarks to the top of a collection | Free |
| Pin Collections | Pin entire collections to the dashboard | Free |
| Favorites / Starred | Star bookmarks for quick access | Free |

---

## 8. Collection Privacy Levels

| Level | Description | Tier |
|---|---|---|
| Private | Only the owner can view | Free |
| Unlisted | Anyone with the link can view | Free |
| Public | Discoverable in the public directory | Free |

---

## 9. Sharing & Collaboration

| Feature | Description | Tier |
|---|---|---|
| Shareable Links | Generate unique URLs for collections | Free |
| View-only Sharing | Recipients can view but not edit | Free |
| Social Sharing | Share to X, Facebook, LinkedIn | Free |
| Collaboration ★ | Invite users to co-manage collections | **Premium** |
| Embed Code ★ | Embed public collections on external websites | **Premium** |

---

## 10. Search & Discovery

| Feature | Description | Tier |
|---|---|---|
| Full-text Search | Search across titles, descriptions, URLs, and tags | Free |
| Filter by Tags | Multi-tag filtering with AND/OR logic | Free |
| Filter by Platform | Show only bookmarks from specific sources | Free |
| Filter by Date | Date range selection | Free |
| Sort Options | By date, title, custom order, most visited | Free |
