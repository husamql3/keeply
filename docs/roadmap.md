# Product Roadmap

**2026 Release Plan · 4 Phases**

---

## Overview

| Phase | Focus | Period | Goal |
|---|---|---|---|
| Phase 1 | Foundation | Q1 2026 (Months 1–2) | Ship a working MVP |
| Phase 2 | Organization & Discovery | Q2 2026 (Months 3–4) | Advanced org tools and filters |
| Phase 3 | Premium & Sharing | Q3 2026 (Months 5–6) | Launch paid subscription |
| Phase 4 | Polish & Scale | Q4 2026 (Month 7+) | Stability and community growth |

---

## Phase 1 — Foundation

**Q1 2026 · Months 1–2**

> **Goal:** Ship a working MVP with core auth, bookmark saving, and basic organization.

- Google & GitHub OAuth + Magic Link login
- Manual bookmark entry (URL, title, description, tags)
- Browser extension / bookmarklet (Quick Add)
- Duplicate URL detection
- Basic Collections: create, list, view
- Grid View and List View layouts
- Full-text search (title, URL, description)
- Private / Unlisted / Public privacy levels
- Shareable collection links

---

## Phase 2 — Organization & Discovery

**Q2 2026 · Months 3–4**

> **Goal:** Enrich bookmark management with advanced org tools and search filters.

- Nested Collections (up to 3 levels)
- Drag-and-drop bookmark reordering
- Move bookmarks between collections
- Kanban View
- Pinning: bookmarks within collections + collections to dashboard
- Favorites / Starred bookmarks
- Multi-tag filtering (AND/OR logic)
- Filter by platform, date, and sort options
- Collection stats (count, last updated)

---

## Phase 3 — Premium & Sharing

**Q3 2026 · Months 5–6**

> **Goal:** Launch premium subscription with collaboration and advanced customization.

- ★ Color-coded Tags
- ★ Collaboration: invite co-managers to collections
- ★ Embed Code for public collections
- Source platform indicator on bookmarks
- Collection cover images (custom or auto-generated)
- Social Media Sharing (X, Facebook, LinkedIn)
- View-only sharing mode
- Public directory / discoverability

---

## Phase 4 — Polish & Scale

**Q4 2026 · Month 7+**

> **Goal:** Stability, performance, and community growth.

- AI-powered bookmark suggestions and auto-tagging
- Import/export (HTML bookmarks, CSV)
- Mobile apps (iOS & Android)
- Team workspaces for organizations
- Analytics dashboard (click tracking, most visited)
- Advanced semantic search
- Public API for power users and integrations

---

*★ = Premium feature · Phases and timelines subject to change based on user feedback.*
